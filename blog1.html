First Blog-17/09/2018         Get Instagram Chats Offline

         This is instagram offline chat save script. I have written this script for those who want to have their instagram chats saved in .txt format.


         This script works completely offline. And it only works for personal chats right now. But I am working to make it for Group chats also.


         You can fork the script if you want from my github account and do changes as you wish. I am making this opensource so that everyone can use it and I can know if there are any flaws in my script.



About InstaChatSave

         Easy to read Instagram messages after downloading json file of the chat.


How To Use:-

         First of all download the chat backup data from your instagram application's settings.

         There will be an option called "Data Download" under privacy and security tab.

         One zip file will be sent over your email. You must download that zip file in your computer/laptop.

         Once that is downloaded and extracted in a folder, download the file which is given here.

         Now copy/cut Messages.py file in the folder where your messages.json file from instagram backup data is.

         Then double click on the Messages.py file and enter the username of the person.

         All the chats will be saved as .txt file in that folder.
