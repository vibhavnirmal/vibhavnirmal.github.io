Instagram Chats Offline17/09/2018


      This is instagram offline chat saving script. I have written this script for those who want to have their instagram chats saved in .txt format.


      This script works completely offline. And it only works for personal chats right now. But I am working to make it for Group chats also.


      You can fork the script if you want from my github account and do changes as you wish. I am making this open source so that everyone can use it and I can know if there are any flaws in my script.



About InstaChatSave :-

Easy to read Instagram messages after downloading json file of the chat.


How To Use:-

  1. First of all download the chat's backup data from your instagram application's settings. If that is not working try in web-browser.
  2. There will be an option called "Data Download" under privacy and security tab.
  3. One zip file will be sent over your email. You must download that zip file in your computer/laptop.
  4. Once that is downloaded and extracted in a folder, download the file which is given here.
  5. Now copy/cut Messages.py file in the folder where your messages.json file from instagram backup data is.
  6. Then double click on the Messages.py file and enter the username of the person.
  7. All the chats will be saved as .txt file in that folder.
  8. If you find anything to be improved, Please Let me know.
