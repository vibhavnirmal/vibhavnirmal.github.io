      I would like to share my experience how I learnt programming. Whenever I have an idea to create a project (or to program a basic script), I just search for the logic which will be needed. And gradually from reading and understanding the logic behind the code, I develope the program. I did not learn coding through reading books. Books can only help us with understanding the logic how specific function works, but in real life we have to make changes to that function and apply our own logic to modify it according to our needs. I prefered Dan Bader's and Sentdex's content to learn basics.
