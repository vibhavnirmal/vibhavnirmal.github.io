Installing Laravel on Ubuntu 20.04

November 18, 2022
Laravel Ubuntu PHP Web Development
View Gist

This comprehensive guide will walk you through installing Laravel v9.40.1 with PHP v8.1.12 on Ubuntu 20.04. Laravel is a powerful PHP framework that makes web development elegant and enjoyable with its expressive syntax and robust features.

Prerequisites: This tutorial assumes you have a fresh Ubuntu 20.04 installation with sudo privileges and basic familiarity with the command line.

Step 1: Add PHP Repository

First, we need to add the Ondřej Surý PPA repository, which provides the latest PHP versions for Ubuntu: 

sudo add-apt-repository ppa:ondrej/php

What this does: This command adds the ppa:ondrej/php repository to your system's package sources. This repository contains various PHP versions and related packages that may not be available in the default Ubuntu repositories.

Step 2: Update Package List

After adding the new repository, update your package list to include the newly available packages: 

sudo apt update

What this does: The apt update command refreshes the package list from all configured repositories, ensuring you have access to the latest versions of available packages.

Step 3: Verify Repository (Optional)

To confirm the repository was added correctly, you can search for a specific PHP package: 

sudo apt-cache search php8.1-curl

Expected output: 

php8.1-curl - CURL module for PHP

Step 4: Install PHP and Required Extensions

Now install PHP 8.1 along with essential extensions needed for Laravel: 

sudo apt install php8.1-cgi php8.1-cli php8.1-xml php8.1-curl

Package breakdown:

  • php8.1-cgi - CGI/FastCGI SAPI for PHP 8.1
  • php8.1-cli - Command-line interpreter for PHP 8.1
  • php8.1-xml - XML processing support for PHP 8.1
  • php8.1-curl - CURL library support for PHP 8.1

Step 5: Install Composer

Composer is PHP's dependency manager and is required to install Laravel: 

sudo apt install composer

What this does: Installs Composer globally on your system, allowing you to manage PHP dependencies and create Laravel projects.

Step 6: Create a New Laravel Project

Use Composer to create a new Laravel project in your desired directory: 

composer create-project laravel/laravel myProject

What this does: Downloads and installs the latest stable version of Laravel in a directory named "myProject". You can replace "myProject" with your preferred project name.

Step 7: Navigate and Start Development Server

Navigate to your project directory and start the Laravel development server: 

cd myProject
php artisan serve

Expected output: 

INFO Server running on [http://127.0.0.1:8000]

Verification

Open your web browser and navigate to http://127.0.0.1:8000 to see the Laravel welcome page, confirming your installation was successful.

Next Steps

Now that Laravel is installed and running, you can:

  • Configure your database connection in the .env file
  • Run migrations to set up your database schema
  • Install additional PHP extensions as needed (e.g., php8.1-mysql, php8.1-mbstring)
  • Set up your preferred code editor with Laravel extensions
  • Begin developing your web application

Troubleshooting Tips

If you encounter any issues:

  • Ensure all PHP extensions are properly installed
  • Check that Composer is accessible globally
  • Verify your system meets Laravel's minimum requirements
  • Check file permissions in your project directory
