Instagram Chats Offline    September 17, 2018

This is instagram offline chat saving script. I have written this script for those who want to have their instagram chats saved in .txt readable format. This script works completely offline. And it only works for personal chats right now. But I am working to make it for Group chats also. You can fork the script if you want from here and do changes as you wish. If you find any issues fork this repo and send a pull request.

How To Use:-

  1. Download the chat's backup data from your instagram application settings
  2. There will be an option called "Data Download" under privacy and security tab
  3. Once you receive the zip file in your mail, download and extract in a folder.
  4. Download python script from here.
  5. Now copy/cut Messages.py file in the folder where your messages.json file from instagram backup data is.
  6. Then double click on the Messages.py file and enter the username of the person.
  7. All the chats will be saved as .txt file in that folder.
