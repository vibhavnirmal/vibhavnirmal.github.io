4D Multi Object Tracking    December 04, 2023

Well you might think, Yeah it looks cool but what is 4D Multi Object Tracking?
4D Multi Object Tracking is a method to track multiple objects in a 3D space over time.
How can we do that?
LiDAR data processing, point rcnn and kalman filter are the key components of this method.
Point RCNN is used to detect objects in the point cloud data.
